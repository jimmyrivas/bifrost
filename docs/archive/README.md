# Archived planning documents

These are **historical planning documents** from Bifrost's early development.
They are kept for context and archaeology, but they are **not** an accurate
description of the current application — several of them state feature status
that a later code audit (2026-07-10) proved unreliable.

For the current, verified truth use:

- [`docs/guide/`](../guide/README.md) — the user guide, verified against the code (EN/ES)
- [`openspec/specs/`](../../openspec/specs/) — the capability specifications
- [`README.md`](../../README.md) — the honest feature tiers

| Document | What it was |
| --- | --- |
| `prompt.md` | The original architectural prompt/spec (Ásbrú successor vision) |
| `IMPLEMENTATION_PLAN.md` | The original ~107-feature plan (Tabby + Ásbrú + DevOps analysis) |
| `STATUS.md` | A per-feature status tracker, last updated 2026-03-22 (v0.2.0) — **do not trust its "done" marks** |
| `COMPLETION_PLAN.md` | A completion plan toward 1.0, superseded by OpenSpec changes |
| `stitch-design-brief.md` | A design brief written for Google Stitch |
